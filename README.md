## 🐍 Fundamentos de Python - Data Science Academy (DSA)

>Este repositório contém os exercícios, projetos, anotações e desafios desenvolvidos durante os estudos do curso Fundamentos de Linguagem Python - Do Básico a Aplicações de IA, da Data Science Academy (DSA).

>O objetivo deste repositório é documentar a evolução do aprendizado, desde os conceitos básicos da linguagem Python até tópicos mais avançados voltados para desenvolvimento de aplicações, análise de dados e Inteligência Artificial.

## 📚 Conteúdo do Repositório
#### 🤖 Projeto Inicial: Chatbot para Dúvidas de Python

>O primeiro projeto desenvolvido foi um chatbot simples capaz de responder perguntas relacionadas à linguagem Python. Durante sua construção foram aplicados conceitos como:

1. Variáveis
2. Estruturas condicionais
3. Loops
4. Funções
5. Manipulação de strings
6. Entrada e saída de dados
7. Estruturas de decisão
